Put short scene clips or animated images here.

Supported formats:
- Video: .mp4, .webm, .mov
- Animated/image: .gif, .webp, .png, .jpg, .jpeg

Example:
1. Add a file such as `velma-kelly.mp4` or `velma-kelly.gif`.
2. In `src/data/personas.ts`, add:

```ts
momentMediaPath: '/moments/velma-kelly.mp4',
momentMediaAlt: 'Velma Kelly scene clip',
```

Use short, compressed clips for mobile pages.
